一.PDI（Kettle）安装

发表于 2018-09-15

1. 创建转换流程

以如下将mongodb库中数据导出到Excel文件的简单过程为主进行讲解

点击图中红框的图标可以指定连接到上一个数据源

选择左上“文件->新建->转换”，如图

2. 拖动创建MongoDB Input数据源，并配置

  1. 从左侧“Big Data”中拖动MongoDB Input
  2. 配置MongoDB连接中的IP,Port,用户名，密码
  3. 配置要查询的数据库和集合
  4. 填写查询条件
  5. 选择如图中的单个json字段，以及输出的json字段名称
  6. 预览查看数据

3.拖动JSON Input，并配置

  1. 从左侧“Input”中拖动JSON Input,并连接到MongoDB Input
  2. 在“文件”Tab中，选择“源定义在一个字段里”，然后“选择字段获取源”中的MongoDB Input 输出的字段名称
  3. 在“字段”Tab中配置要获取的字段名称及路径，类型

4.拖动Excel输出，并配置

  1. 从左侧“输出”中拖动Excel 输出并配置
  2. 在”文件”Tab中配置输出文件名称，路径

  1. 在“字段”Tab中，点击获取字段，配置要输出的字段及类型

5. 运行查看数据
